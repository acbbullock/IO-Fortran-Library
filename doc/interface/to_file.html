public interface to_file

Subroutine for writing an array of uniform data type to an external file.

The file file_name will be created if it does not already exist and will be overwritten if it does exist. Writing to text is allowed for arrays of rank 1 or 2, and writing to binary is allowed for arrays of any rank 1-15. Any invalid actual arguments will be ignored, defaults will be assumed, and a warning message will be issued on stdout.

For a user reference, see to_file.
